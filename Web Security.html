Web Security

Here at ECI we care about our customers and one thing that we like to share is information. With the web being so commonly used today it is important that you understand what web security is. Below are just a few examples of properties that are obtained from the user to allow better user side browsing of web pages.

To demonstrate even more how much information is shared on a website below will show your system information.

The

  • Width
    • Width gathers infromation about a users display screen width to allow the user friendly adjusted screen size.
  • Height
    • Height also is used to gather information from a users screen but to find height to adjust the screen height.
  • Platform
    • The platform is used to gather information about the users operating system.
  • Online
    • The online property detects if the users browser is currently connected to the internet.
  • User Agent
    • The user agent property obtains information about the browser, platform name, and compatibility.
  • Pixel Depth
    • This propety obtains information on the users display screen color resolution in bits per pixel.
